<PageDescription>

The **Update Project** allows user to: 1) edit the project's profile, 2) finalize the project, and 3) submit signed copy of the project profile.

</PageDescription>

## Navigation

To get to the update project page, user must open draft, updated or returned projects. First, navigate to the **Projects** page. Then, click on either **Created**, **Updated** or **Returned** buttons.

The image below shows the list of projects that can be updated. Click on the **dropdown** button to show the options. Finally, click on **Update**.
